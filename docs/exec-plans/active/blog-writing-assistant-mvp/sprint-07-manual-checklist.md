# Sprint 07 Manual Checklist: 네이버 붙여넣기 검증

## 사전 조건

- 앱 실행: `npm run dev`
- 생성 페이지에서 본문이 최소 2개 문단 이상 존재
- 네이버 블로그 에디터(신규 글 작성 화면) 열림

## 체크리스트

1. **전체 글 네이버 복사**
   - 앱에서 `전체 글 네이버 복사` 클릭
   - 네이버 에디터에 붙여넣기
   - 제목/문단/리스트 기본 서식이 유지되는지 확인

2. **선택 문단 복사**
   - BlockNote에서 일부 문단만 드래그 선택
   - `선택 문단 복사` 클릭 후 네이버 에디터에 붙여넣기
   - 선택한 문단만 붙여넣어지는지 확인

3. **선택 없음 가드**
   - 문단 선택 없이 `선택 문단 복사` 클릭
   - `선택된 문단이 없습니다...` 안내 메시지가 표시되는지 확인

4. **Markdown 복사 fallback**
   - `Markdown 복사` 클릭
   - 텍스트 에디터에 붙여넣어 Markdown 원문이 들어오는지 확인

5. **변환 fallback 확인**
   - 변환이 어려운 특수 블록을 포함한 본문으로 복사
   - 붙여넣기 결과가 완전히 비지 않고 최소 텍스트가 유지되는지 확인
